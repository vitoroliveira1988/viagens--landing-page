# Viagens - Landing Page

Este projeto é uma página de captura (Landing Page) simples sobre turismo, focada em vilarejos europeus.

## Tecnologias Utilizadas
- HTML5 (Semântico)
- CSS3 (Flexbox e Efeitos Hover)

## Como executar
1. Baixe os arquivos `index.html` e `style.css`.
2. Abra o arquivo `index.html` em qualquer navegador moderno.

## Funcionalidades
- Navegação interna através de âncoras (ID's).
- Efeito Hover nos botões e menus.
- Design responsivo e limpo.
